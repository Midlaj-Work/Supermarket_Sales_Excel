# 📊 Supermarket Sales Dashboard Using Microsoft Excel

An interactive Microsoft Excel dashboard built to analyze supermarket sales performance using Pivot Tables, Pivot Charts, Slicers, and KPI Cards. The dashboard provides business insights into sales, profit, customer behavior, payment methods, product categories, and city-wise performance.

---

## 📌 Project Overview

This project analyzes a real-world supermarket sales dataset and transforms raw transactional data into an interactive business dashboard. The dashboard helps monitor key business metrics and supports data-driven decision-making through dynamic filtering and visualizations.

---

## 🎯 Objectives

* Analyze overall sales performance
* Monitor total revenue and profit
* Compare product line performance
* Analyze customer type and gender distribution
* Study payment method preferences
* Compare sales across different cities
* Build an interactive Excel dashboard
